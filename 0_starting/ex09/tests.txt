================================================================================
                        DIFERENCES BETWEEN USE "tar.gz" or "whl"
================================================================================


pip install ft_package-0.0.1.tar.gz
# 1. Extract source
# 2. Run setup.py/pyproject.toml  
# 3. Build metadata
# 4. Install
# → Slower

pip install ft_package-0.0.1-py3-none-any.whl
# 1. Extract and copy files
# → Much faster!
