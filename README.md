# 42 - Python for Data Science

Initial modules covering data science fundamentals through hands-on exercises.

## Modules

**Module 0: Starting**  
Python basics - data structures, functions, command-line processing, list comprehensions, generators, and package creation.

**Module 1: Array**  
NumPy arrays, image manipulation, slicing operations, and color filter implementations.

**Module 2: DataTable**  
Loading, manipulating, and displaying datatables.

**Module 3: OOP**  
Object-Oriented Programming principles and implementation.

**Module 4: DoD**  
Data-Oriented Design patterns and practices.

## Standards

- PEP 8 compliant (flake8)
- Complete documentation
- Proper error handling
- No global scope code
